## ⚙️ Shell Scripts Included

### 1️⃣ System Identity Report

**File:** `sys_identity.sh`

* Kernel version
* OS distribution
* User info
* Uptime
* License details

---

### 2️⃣ FOSS Package Inspector

**File:** `package_inspector.sh`

* Checks if python3 is installed
* Works with dpkg / rpm
* Displays FOSS philosophy message

---

### 3️⃣ Disk & Permission Auditor

**File:** `disk_auditor.sh`

* Scans system directories
* Shows permissions, owner, size
* Includes Python library audit

---

### 4️⃣ Log Analyzer

**File:** `log_analyzer.sh`

* Reads log file
* Counts keyword occurrences
* Displays last 5 matching lines

---

### 5️⃣ Open Source Manifesto Generator

**File:** `manifesto.sh`

* Takes user input
* Generates personal manifesto
* Saves output in `.txt` file
